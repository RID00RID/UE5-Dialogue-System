# UE5 Dialogue System Plugin

Плагин системы диалогов для Unreal Engine 5, созданный в учебных целях.  
На текущий момент это базовая реализация, которая может служить фундаментом для собственной доработки и расширения.

> ⚠️ Плагин не является завершённым production-решением и содержит ряд недочётов.

---

## 📌 Описание

Данный плагин предоставляет простую и расширяемую систему диалогов с графическим редактором, поддержкой кастомной логики и интеграцией в UI.

Он подойдёт как:
- стартовая точка для собственной системы диалогов
- учебный пример архитектуры диалоговых систем
- основа для расширения под конкретный проект

---

## ✨ Возможности

- 🧩 **Графический редактор диалогов**
  - Удобное создание диалоговых веток
  - Визуальная работа с репликами и связями

- ⚡ **Кастомные события**
  - Создание собственных событий в редакторе
  - Вызов событий во время выполнения диалога

- 🔀 **Conditions (условия)**
  - Добавление логики для ветвления диалогов
  - Гибкое управление поведением

- 🎯 **События при выборе реплики**
  - Триггеры при выборе вариантов ответа
  - Интеграция с игровыми системами

- 🖥️ **Простая UI-обёртка**
  - Базовый интерфейс для отображения диалогов
  - Возможность кастомизации под проект
